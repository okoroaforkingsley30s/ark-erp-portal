ARK ONE 1.0.16 — COMPLETE ARK MAIL REPAIR
Date: 2026-07-22

This cumulative repair covers:
- private per-user mailbox access and hidden Google tokens
- paginated Gmail sync including Spam and Trash
- Inbox, Starred, Snoozed, Sent, Drafts, Scheduled, All Mail, Spam and Trash
- read/unread, star/unstar, archive, snooze, spam, trash, restore and permanent delete
- correct Reply, Reply All, Forward, CC and BCC handling
- outgoing and incoming attachment support
- Google Drafts and scheduled Google delivery
- automatic OAuth popup return and mailbox refresh
- one conversation row in the list with the complete thread in the reader
- Helpdesk-only atomic mail-to-ticket conversion
- Google-only major notification delivery (no competing Resend path)

COMPANY POLICY PRESERVED
Only Helpdesk can create service tickets and convert client mail to tickets.
This repair does not give Business Development, System Admin or another role
permission to create or close Helpdesk calls.

INSTALL ON A NEW GIT BRANCH

1. Extract this ZIP into the repository root with overwrite enabled.

2. Validate:

  npm run lint
  npm run typecheck
  npm test
  npm run build
  node scripts/validate-migrations.mjs
  npx supabase db push --linked --dry-run

The dry run must list only:

  202607220010_ark_mail_complete_mailbox.sql

3. Review, commit, push and open a pull request. Wait for all 6/6 checks before
merging. Do not add .env files, Supabase temp files or production backups.

PRODUCTION DEPLOYMENT (AFTER MERGE)

4. Capture a fresh production schema and data checkpoint first.

5. Apply the database migration:

  npx supabase db push --linked

6. Deploy the changed/new functions:

  $projectRef="fryidzyhqhdenghyxjfp"

  npx supabase functions deploy gmail-oauth --project-ref $projectRef
  npx supabase functions deploy gmail-sync --project-ref $projectRef
  npx supabase functions deploy gmail-send --project-ref $projectRef
  npx supabase functions deploy gmail-reply --project-ref $projectRef
  npx supabase functions deploy gmail-action --project-ref $projectRef
  npx supabase functions deploy gmail-draft --project-ref $projectRef
  npx supabase functions deploy gmail-schedule --project-ref $projectRef
  npx supabase functions deploy send-notification-email --project-ref $projectRef
  npx supabase functions deploy dispatch-notification-outbox --project-ref $projectRef --no-verify-jwt

7. Confirm the existing dispatch-notification-outbox schedule still runs every
minute. It now delivers both major notifications and due scheduled mail through
approved connected Google company mailboxes.
